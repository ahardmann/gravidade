Objetivo

Desenvolver um jogo para plataforma Android utilizando as ferramentas Android Studio e o framework AndEngine.

Especificação


Utilizar o Android Studio, AndEngine framework, SQLite e GitHub.
Implementar: colisão, três fases ou três níveis de dificuldade e interação.

Referência Teórica

  • Análise Orientada à Objetos
  • Banco de Dados II
  • Programação Estruturada
  • Programação Orientada à Objetos

Projeto

Gravidade é um jogo que mescla dois universos:
A nostalgia dos jogos de nave como o thunder force e com os jogos atuais para celular como o flappy bird, trazendo consigo um grau de dificuldade elevado e desafiador.
